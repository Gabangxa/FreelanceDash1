Configuration Steps
  1. Go to the Polar.sh Developer Portal
  2. Create a new OAuth Application
  3. Enter your application name (e.g., "Freelancer Suite")
  4. Set the Redirect URI to the value shown above
  5. After creating the application, you'll receive:
    • Client ID
    • Client Secret
  6. Add these values to your application's environment variables as:
    • POLAR_CLIENT_ID
    • POLAR_CLIENT_SECRET
Once configured correctly, your payment system will connect to Polar.sh securely for subscription management.
